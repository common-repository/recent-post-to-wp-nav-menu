A WordPress plugin to include the latest post link for a post type to the nav menu.

== Description ==
Simple, light weight plugin for WordPress to include the recent post link for a post type to the menu.

== Installation ==
1. Search for 'recent post to wp nav menu' in WordPress Plugin Directory and Download the .zip file & extract it.
2. Upload 'recent-post-to-wp-nav-menu' folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Navigate to the 'Menus' menu in WordPress.
